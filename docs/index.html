Your pocket OS, without the scavenger hunt

The safe, one-window Slax USB maker.

SlickSlax handles MBR, FAT32, copying, boot setup, and verification in one guided pass. It does not raw-flash the ISO — Slax needs a writable filesystem for persistent sessions.

What it does

  1. Identifies a removable USB device
  2. Creates MBR + FAT32 layout
  3. Copies the Slax /slax tree from your ISO
  4. Installs boot files and optional persistence
  5. Verifies critical assets before eject
