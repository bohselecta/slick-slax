# Security policy

SlickSlax performs privileged, destructive disk operations. Please do not publish a credible drive-targeting or command-injection vulnerability before a fix is available.

Report security issues privately through GitHub’s **Report a vulnerability** feature for this repository. Include the operating system, SlickSlax version, reproduction steps, and whether any data was lost.

The project does not collect telemetry or transmit ISO paths, drive identifiers, or installation history.
